# l2wol &mdash; despatch Wake-on-LAN magic packets on Layer 2

## How to Build

```console
% cc -o l2wol l2wol.c
```


## Usage

May require privilege to perform.

```plain
l2wol [-b] [-i interface] destination ...
```


## Note

:warning: The source code is not compatible with Linux.
